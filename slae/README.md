# Лабораторная работа №1: Решение систем линейных алгебраических уравнений

Проект реализует набор численных методов для решения систем линейных алгебраических уравнений (СЛАУ) из лабораторной работы №1.

## Реализованные методы

- **LU-разложение**
- **Метод простых итераций**
- **Метод Якоби**
- **Метод Гаусса-Зейделя**
- **Метод наискорейшего спуска** (для симметрично положительно определённых матриц)
- **Метод сопряжённых градиентов** (для симметрично положительно определённых матриц)

## Как пользоваться `main.py`

1. Запустите скрипт:
   ```bash
   python3 main.py
   ```
2. Введите размер системы `n`.
3. Введите строки матрицы `A`, разделяя элементы пробелами.
4. Введите элементы вектора правых частей `b`.
5. Выберите метод решения из списка. Для некоторых методов потребуются дополнительные параметры (точность, максимум итераций, параметр `tau`).
6. Получите решение, которое выводится построчно.

## Тестирование и покрытие

Запуск тестов:
```bash
python3 -m unittest discover
```

Получение отчёта по покрытию (используется `coverage`):
```bash
python3 -m coverage run -m unittest discover
python3 -m coverage report -m solver.py
```

Текущее покрытие на момент последнего прогона — **100%** для `solver.py`.
